# NASA Space Explorer

A responsive static website showcasing NASA space missions, planets, gallery, and more. Built with HTML, CSS, JavaScript, and PWA features (service worker included).

## Features
- Home page with space-themed animations
- About NASA
- Missions overview
- Planets information
- Image gallery
- Progressive Web App (PWA) support for offline use

## How to Run
1. Open `index.html` in a modern web browser.
2. For best experience, serve via a local server (e.g., `npx serve` or VS Code Live Server extension).
3. Install as PWA from the browser for offline access.
